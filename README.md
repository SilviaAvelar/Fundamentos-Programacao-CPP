# 🚀 Jornada de Fundamentos em C/C++

Este repositório é o registro da minha evolução técnica de **Fundamentos de Programação**.
Aqui, transformei lógica abstrata em sistemas funcionais, explorando desde a base da Linguagem C até recursos avançados do C++.

## 🛠️ Tecnologias e Ferramentas
- **Linguagens:** C e C++
- **IDE:** Visual Studio 2022
- **Ambiente:** Windows Console Application (Codepage 1252/65001)

## 📂 Estrutura do Repositório
O projeto está organizado por marcos de aprendizado:

- **`01-Conceitos-Basicos`**: Anatomia de um programa, tipos de dados (`int`, `float`, `char`) e operadores aritméticos.
- **`02-Estruturas-de-Decisao`**: Lógica condicional complexa, `if/else` aninhados e seletores `switch case`.
- **`03-Estruturas-de-Repeticao`**: Domínio de loops (`for`, `while`, `do-while`) e **Validação Robusta de Dados** (tratamento de buffer).
- **`04-Arrays-e-Matrizes`**: Manipulação de coleções de dados, vetores dinâmicos e tabelas bidimensionais.
- **`05-Modularizacao-Funcoes`**: Organização de código em blocos reutilizáveis e o segredo da **Passagem por Referência (`&`)**.
- **`06-Projetos-Consolidacao`**: Aplicações finais que unem todos os pilares anteriores em sistemas reais.

## 🏆 Projetos em Destaque

### 🗳️ Sistema de Apuração Eleitoral
Um simulador completo de votação que processa dados via Arrays, utiliza o operador ternário para otimização e possui uma lógica automática de desempate entre candidatos.

### ☕ Gestão de Cafetaria Modular
Um sistema de pedidos interativo que demonstra o uso de funções para manter um código limpo e escalável, utilizando passagem por referência para gerir a memória de forma eficiente.

## 🧠 Principais Aprendizados
- **Código Limpo:** Foco em nomes de variáveis claros e documentação de cabeçalho.
- **Programação Defensiva:** Implementação de verificações para garantir que o programa não interrompa por erros de entrada do utilizador.
- **Eficiência de Memória:** Entendimento da diferença técnica entre valor e endereço de memória.
